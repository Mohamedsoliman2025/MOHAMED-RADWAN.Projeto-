# Laudo de Qualidade — Projeto PIT II

## Introdução

Este laudo apresenta a análise de qualidade da aplicação desenvolvida por Mohamed Radwan, com base em testes realizados por usuários e validações técnicas.

## Testes Realizados

- Testes manuais no front-end (formulário, validação, responsividade)
- Testes automatizados com JUnit no back-end
- Testes com 5 colegas documentados no formulário

## Erros Identificados

| Erro | Correção Aplicada |
|------|-------------------|
| Layout quebrado em telas pequenas | Ajuste com media queries |
| Botão de logout sem resposta | Correção do evento de clique |
| Tempo de carregamento elevado | Otimização de imagens e lazy loading |

## Funcionalidades Corrigidas

- Validação de campos obrigatórios
- Integração entre front-end e back-end
- Persistência de dados no MySQL

## Evidências

As evidências visuais (prints antes/depois) estão disponíveis na pasta `/assets/evidencias`.

## Conclusão

A aplicação está funcional, com os principais erros corrigidos e validada por usuários reais. Está pronta para entrega acadêmica e futuras melhorias.
