# Manual do Usuário — Projeto PIT II

## Introdução

Este manual tem como objetivo orientar o usuário sobre o uso da aplicação desenvolvida por Mohamed Radwan no Projeto Integrador Transdisciplinar em Engenharia de Software II.

## Acesso à Aplicação

- **URL:** [https://projeto-pit2.onrender.com](https://projeto-pit2.onrender.com)
- Requisitos: Navegador atualizado (Chrome, Firefox, Edge)

## Funcionalidades

### 1. Cadastro de Usuário
- Acesse a página inicial
- Preencha os campos "Nome" e "Email"
- Clique em "Salvar"
- Uma mensagem de sucesso será exibida

### 2. Listagem de Usuários
- Os usuários cadastrados são exibidos automaticamente após o envio
- Atualize a página para ver novos registros

## Regras de Uso

- Campos obrigatórios: Nome e Email
- Emails devem estar em formato válido (ex: usuario@dominio.com)
- Não é permitido duplicar emails
